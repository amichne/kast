# Kast for Codex

Use `kast agent` and scoped `--help` to discover compiler-backed Kotlin and Gradle operations.

Mutations plan, apply, and validate diagnostics synchronously. Treat exit code 0 as terminal success. On failure, report only the structured error code, message, and actionable details returned by Kast.
