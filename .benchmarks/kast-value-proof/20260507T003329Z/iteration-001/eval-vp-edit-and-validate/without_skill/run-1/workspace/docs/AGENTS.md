# Scope

- This file applies to `docs/`.

## Work here

- Validate with `make docs-build`.
- Preview locally with `make docs-serve`.

## Edit rules

- Edit Markdown, CSS, and JS in this tree.
- For navigation, site metadata, or theme settings, edit `../zensical.toml`.
- Do not edit `docs/venv/` or `../site/`; they are local environment or generated site output.
